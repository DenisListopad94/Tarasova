git init: Инициализирует новый репозиторий Git в текущем каталоге
git clone: Создает копию удаленного репозитория на вашем локальном компьютере
git add: Добавляет измененные файлы в индекс, подготавливая их к коммиту
git commit: Создает новый коммит с сохраненными изменениями в индексе
git push: Отправляет ваши коммиты в удаленный репозиторий
git pull: Получает изменения из удаленного репозитория и объединяет их с вашим текущим рабочим деревом
git branch: Показывает список веток в репозитории и текущую активную ветку
git checkout: Переключается между ветками или коммитами
git merge: Объединяет изменения из одной ветки в другую
git rebase: Переносит изменения из одной ветки в другую, изменяя историю коммитов
git cherry-pick: Применяет отдельный коммит из одной ветки в другую
git status: Показывает текущее состояние рабочего дерева и индекса
git log: Показывает историю коммитов в репозитории
git reset: Отменяет коммиты или изменения в индексе
git revert: Создает новый коммит, который отменяет изменения определенного коммита
git stash: Сохраняет временные изменения, чтобы можно было переключиться на другую ветку или восстановить их позже
git diff: Показывает различия между двумя состояниями репозитория
